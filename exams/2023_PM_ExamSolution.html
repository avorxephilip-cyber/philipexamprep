Question 4

A.. Define project integration management. [(2 marks)] Model Answer: Project Integration Management involves the processes and activities to identify, define, combine, unify, and coordinate the various processes and activities within the Project Management Process Groups. It includes making choices about resource allocation and managing the interdependencies among all project management knowledge areas. 📖 Source: Unit 5, Session 1 — Project Integration Management, p.119 B.. Explain the processes involved in project integration management. [(12 marks)] Model Answer: Develop Project Charter: Formally authorizes the existence of the project and provides the project manager with authority to apply organizational resources to project activities. Develop Project Management Plan: Defines, prepares, and coordinates all subsidiary plans and integrates them into a comprehensive PMP that describes how the project will be executed, monitored, controlled, and closed. Direct and Manage Project Work: Leading and performing the work defined in the PMP and implementing approved changes to achieve project objectives. Manages team, resources, communications, and risks. Monitor and Control Project Work: Tracking, reviewing, and reporting project progress against performance objectives defined in the PMP. Determines corrective or preventive actions when deviations occur. Perform Integrated Change Control: Reviewing all change requests and approving or rejecting changes to deliverables, project documents, and the PMP. Reduces project risk by integrating documented changes. Close Project or Phase: Finalizing all activities to formally complete the project or phase, releasing resources, and transferring deliverables to the customer. 📖 Source: Unit 5, Session 1 — Project Integration Management Process, pp.119–120 C.. State two importance of project integration management. [(3 marks)] Model Answer: Initiates Project by Allocating Resources: Managing integration helps initiate the project by allocating resources to commence work. The project charter directs teams to execute the project plan and creates a formal record with senior management commitment. Prevents Isolated Project Management: Integration management ensures projects are not managed in isolation — it accounts for how aspects of the project relate to each other and to other parts of the organization, ensuring holistic execution. 📖 Source: Unit 5, Session 1 — Importance of Project Integration Management, p.120 D.. State two challenges in project integration management. [(3 marks)] Model Answer: Tracking Interdependencies: Keeping track of everything and knowing how one element impacts others is very challenging. If not done accurately, it can lead to project failure — especially in large, complex projects. Information Dissemination in Global Teams: Projects with teams spread globally face difficulties regarding information dissemination. Wrong information may be communicated due to noise along the channel of communication. Technology and strong communication skills must be deployed to minimize this problem. 📖 Source: Unit 5, Session 1 — Challenges of Project Integration Management, p.120
